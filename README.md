# Dax


## Setup

``` bash
npm install
```

## Build

``` bash
npm run build
```
``` bash
npm run build:dev
```
``` bash
npm run build:prod
```

## Run

``` bash
npm run electron
```

### build angular and run in browser only, without electron (dev mode):
``` bash
npm run ng:serve
```


## Package

``` bash
npm run electron:build
```
